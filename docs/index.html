接地型キエーロ組立説明書

地面に直接配置するタイプのキエーロを作成します。

まず本体を先に組み立てます。次に屋根の部分を作成します。

それぞれのパーツの説明

本体パーツの説明

  • 角材（長）2本

    • 本体の背面側で屋根と繋がる柱になります

  • 角材（短）6本

    • 地面に埋まる足になります

  • 杉板（長）2セット

    • 前面と背面を構成します。
    • 24cm幅の杉板を上下につなぎ合わせて使用します。

  • 杉板（短）2セット

    • 左面と右面を構成します。
    • 24cm幅の杉板を上下につなぎ合わせて使用します。

屋根パーツの説明

  • 角材（長）2本

    • 正面から見て枠のヨコ向き部分を構成します

  • 角材（短）3本

    • 正面から見て枠をタテ向き部分を構成します

  • ポリカ波板 1セット

    • 何枚か張り合わせて屋根を構成します。
    • クリアのプラスチック素材です。
    • 屋根を閉じたときに溝の向きが縦になるように取り付けましょう。

その他の金具

  • ボルト＆蝶ナット

    • 本体と屋根を連結させます。
    • 本体背面の角材（長）と、屋根側面の角材（長）にボルトが通るサイズの穴を貫通させましょう。
    • 台風や嵐が予想されるときはボルトを抜いて屋根を取り外しておいて下さい。

  • あおり止め

    • 屋根が風で開かないように、本体につなぎとめる役割をします。
    • 重みでロックが掛かる向きに取り付けて下さい。

  • 傘釘

    • ポリカ波板を屋根に固定する専用の釘です。
    • 必ず、ポリカ波板にはドリルで下穴を開けてから傘釘を打ちつけて下さい。
    • 傘釘はハンマーで打ちつけることができます。

本体を組み立てる

まずは両側面のパーツを組み立てましょう。

板に角材を固定していきます。

左右の側面パーツを組み立てたら、次に前後面のパーツを組み立てます。

図面を見るとわかるように、前面パーツでは脚となる角材を板の端から角材一本分をずらして固定します。

後面パーツも同様に、板の橋から一本分ずらした位置に固定します。また、後面パーツは角材が上下に飛び出すように取り付けます。足の長さを左右側面パーツに合わせて固定しましょう。

屋根を組み立てる

屋根の組み立てで注意する点は、本体への取り付け部分です。屋根パーツを組み立てるときに、本体背面の柱と同じ幅になるように組み立てて下さい。

交差部分にドリルで穴を開けて、ボルトと蝶ナットで取り付けます。

ポリカ波板を装着する

屋根パーツに、ポリカ波板を傘釘で取り付けます。

  1. 溝に沿って雨が流れ落ちる向きで、ポリカ波板を屋根パーツに乗せます。
  2. 傘釘を打ち込む場所にサインペンでマーキングをします。
  3. マーキングした位置にドリルで下穴をあけます。
  4. 屋根パーツに傘釘を打ち込み、波板を固定します。

さいごにあおり止めを取り付けて完成です。おつかれさまでした。

注意点

  • 杉板や角材はそのままだと地面に触れている箇所から腐食していきます。表面処理は焼き杉や防腐塗料などお好みに応じて施して下さい。
  • 台風など強い風が予想されるときは屋根を取り外しておいて下さい。
  • 生ゴミを投入される際は細かく刻んでおくと消化が促進されます。
